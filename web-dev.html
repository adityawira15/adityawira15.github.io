Jawaban

Informasi yang paling ingin saya sampaikan adalah hacktiv8 benar benar is the best, saya yang baru menjalani phase nol sudah merasakan perubahan yang amat besar seperti saya yang dulu tidak mengerti tentang bahasa pemrograman, lalu baru 1 minggu saya menerima materi dari hacktiv8 yang mendorong saya jauh untuk mempelajari tentang bahasa pemrograman dan memaksa saya harus bisa tanpa membuat saya tertekan dengan materi yang begitu banyaknya akhirnya saya bisa mengenal apa itu pemrograman dan saya bisa memperdalam ilmu pemrograman dengan singkat, tinggal gimana saya menerima dan berjuang keras untuk memperdalam ilmu pemrograman yang di berikan oleh guru-guru hacktiv8 yang profesional.
  3. Pertama buat struktur html dasar seperti html, head,title, body. lalu masukan teks isinya terlebih dahulu jika sudah edit tampilan menggunakan css, bisa berbeda file atau bisa juga dalam 1 file html jika css berbeda file maka menggunakan format .css lalu impor style ke dalam file html. setelah itu cek terlebih dahulu html yang di buat buka pada web browser sebelum di commit dan push pada github, jika sudah add, commit dan push file html (jika ada file css tambahkan file css) sebelum di push buat remote untuk terhubung ke webset github lalu jika filenya baru pertama kali di push maka harus memberi perintah git push origin master, jika sudah pernah mempush file tersebut maka jika ingin mempushnya kembali tinggal memberi perintah git push, kita juga bisa mengedit file di dalam github dan mengambil perubahan ke komputer kita dengan memberi perintah git pull.
